# Week 4 - HTML Fundamentals

This folder contains HTML practice files covering basic web page structure and elements.


### Tasks:
 — Basic HTML page with:
  - Paragraph with bold text
  - Heading (H1)
  - Lorem ipsum content
  - External link to Google
  - Unordered list with fruits

## Topics Covered:
- HTML document structure (`<!DOCTYPE>`, `<html>`, `<head>`, `<body>`)
- Text formatting (`<b>`, `<p>`, `<h1>`)
- Links (`<a>`)
- Lists (`<ul>`, `<li>`)
- Basic meta tags and viewport setup
